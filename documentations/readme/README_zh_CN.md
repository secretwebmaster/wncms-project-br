# WNCMS

![WNCMS Dashboard](/documentations/images/wncms-dashboard-zh-cn.webp)

WNCMS 是为了解决创作者、开发者与商家在建站过程中遇到的实际问题，而打造的一套现代化、多语系、高扩充性的 CMS。系统基于 Laravel 12 开发，采用干净的架构、模组化设计以及可扩充的生态，能满足 2025 年以后的网站与系统需求。

WNCMS 的设计目标，是在以下几点之间取得最理想的平衡：

- 初学者也能轻松上手  
- 开发者拥有足够的弹性与扩充能力  
- 正式网站运行时保持高效能  
- 设计师与接案公司能快速自订、快速交付  

无论是部落格、企业官网、着陆页系统、多网域平台，甚至是需要自订资料模型的应用，WNCMS 都能提供稳定且可预期的基础，避免许多传统 CMS 的限制。

---

## WNCMS 为什么会诞生

WNCMS 最初是我自用的专案，起因于市面上找不到符合需求的系统。我需要的是：

- 基于 Laravel 的干净、现代化架构  
- 能同时支援多语系、多网站、多网域  
- 一个比 WordPress 更易于维护与理解的主题框架  
- 稳定、安全且可维护的外挂系统  
- 能作为 SaaS 底层的 CMS  
- 一个后台能统一管理多种自订资料模型（文章、连结、选单、广告等）  

市面上的工具要嘛太臃肿、要嘛不易扩充，也有许多并不适合长期维护或多人协作。因此我决定从零开始设计一套可预期、可扩充又易用的 CMS。

如今，WNCMS 已经在多个正式网站、商用主题、SaaS 专案与影音平台上稳定运行。

## WNCMS 提供什么

### 给一般使用者

- 清晰易懂的后台介面  
- 可安装或自订的前台主题  
- 文章管理  
- 连结管理  
- 广告管理  
- 选单编辑  
- 完整多语系内容支援  

### 给开发者

- Laravel 12 生态完整支援  
- Manager 架构资料模型  
- Models / Controllers / Views 可覆写  
- 主题开发简单明确  
- 外挂开发轻量易懂  
- 快速生成 API  
- 可翻译模型  
- 统一快取与查询流程
