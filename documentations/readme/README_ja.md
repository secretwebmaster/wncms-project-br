# WNCMS

![WNCMS Dashboard](/documentations/images/wncms-dashboard-ja.webp)

WNCMS は、クリエイター・開発者・事業者がサイト構築の過程で直面する実際の課題を解決するために作られた、現代的で多言語対応、そして高い拡張性を備えた CMS です。システムは Laravel 12 をベースに構築され、クリーンなアーキテクチャ、モジュール設計、拡張可能なエコシステムを採用し、2025 年以降のウェブやアプリケーション環境に対応できるよう設計されています。

WNCMS は、次の点のバランスを最適に保つことを目標としています。

- 初心者でも扱いやすい  
- 開発者には十分な柔軟性と拡張性を提供  
- 本番環境でも高いパフォーマンスを維持  
- デザイナーや制作会社が短時間でカスタマイズ・納品できる  

ブログ、企業サイト、ランディングページ、多ドメイン構成のプラットフォーム、さらには独自モデルを必要とするアプリケーションまで、WNCMS は安定した予測可能な基盤を提供し、従来の CMS にありがちな制限を避けられます。

---

## WNCMS が誕生した理由

WNCMS はもともと自分自身のためのプロジェクトとして始まりました。市販のツールでは満足できず、次のような要件が満たされなかったためです。

- Laravel ベースのクリーンで現代的なアーキテクチャ  
- 多言語・複数サイト・複数ドメインを同時に扱える仕組み  
- WordPress よりも保守しやすく理解しやすいテーマ構造  
- 安定して安全、そして長期的に運用しやすいプラグインシステム  
- SaaS の基盤としても利用できる CMS  
- 複数のカスタムデータモデル（記事、リンク、メニュー、広告など）を後端で統一管理できる環境  

既存の CMS は重すぎたり拡張しづらかったり、チーム開発や長期運用に向かないものが多く、自分が本当に使いたい CMS をゼロから設計することにしました。

現在、WNCMS は正式サイト、商用テーマ、SaaS サービス、動画プラットフォームなど多くの実運用環境で使用されています。

## WNCMS が提供するもの

### 一般ユーザー向け

- 直感的でわかりやすい管理画面  
- インストールまたはカスタマイズ可能なフロントテーマ  
- 記事管理  
- リンク管理  
- 広告管理  
- メニュー編集  
- 完全な多言語コンテンツ対応  

### 開発者向け

- Laravel 12 エコシステムのフルサポート  
- Manager ベースのデータモデル構造  
- Models / Controllers / Views の上書きが可能  
- シンプルで明確なテーマ開発  
- 軽量で理解しやすいプラグイン開発  
- API の迅速な生成  
- 翻訳対応モデル  
- 統一されたキャッシュ・クエリ処理フロー
