# WNCMS

![WNCMS Dashboard](/documentations/images/wncms-dashboard-en.webp)

WNCMS is a modern, multilingual, and highly extensible CMS created to solve real-world problems encountered by creators, developers, and businesses. It is built on Laravel 12 with a clean architecture, modular design, and an extensible ecosystem designed for the needs of websites and applications beyond 2025.

The goal of WNCMS is to strike the ideal balance between:

- Being easy for beginners to use  
- Providing flexibility and extensibility for developers  
- Maintaining high performance in production environments  
- Allowing designers and agencies to customize and deliver projects efficiently  

Whether you are building a blog, a corporate website, a landing-page system, a multi-domain platform, or a custom data-driven application, WNCMS provides a stable and predictable foundation without the limitations of traditional CMS platforms.

---

## Why WNCMS Was Created

WNCMS originally began as a personal project because I could not find a system that met all my needs. I needed:

- A clean and modern Laravel-based architecture  
- Built-in support for multi-language, multi-site, and multi-domain  
- A theme framework that is easier to maintain than WordPress  
- A stable, secure, and maintainable plugin system  
- A CMS that could serve as the foundation of SaaS products  
- A unified backend interface to manage multiple custom data models (posts, links, menus, ads, etc.)  

Most existing tools were either too bloated, too rigid, or not suitable for long-term maintenance or team collaboration. Because of that, I decided to build a CMS from scratchâ€”one that is predictable, extensible, and enjoyable to use.

Today, WNCMS runs on production websites, commercial themes, SaaS platforms, and video platforms.

## What WNCMS Offers

### For Regular Users

- Clear and user-friendly admin panel  
- Installable or customizable frontend themes  
- Post management  
- Link management  
- Advertisement management  
- Menu editing  
- Full multi-language content support  

### For Developers

- Full support for the Laravel 12 ecosystem  
- Manager-based data model architecture  
- Overridable Models / Controllers / Views  
- Simple theme development  
- Lightweight plugin system  
- Rapid API generation  
- Translatable model support  
- Unified caching and query flow
